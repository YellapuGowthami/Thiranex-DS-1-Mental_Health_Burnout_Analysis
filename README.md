# Mental Health & Burnout Analysis

This project focuses on data cleaning, preprocessing, and visualization using Python libraries such as Pandas, Matplotlib, and Seaborn.

## Features
- Missing value checking
- Duplicate removal
- Data visualization
- Burnout analysis
- Correlation heatmap

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Project Outcome
This project helped analyze employee burnout patterns and visualize important insights related to stress, sleep, work mode, and work-life balance.
